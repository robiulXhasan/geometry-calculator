DOM is the short form of Document Object Model. It defines all the Html elements as objects, properties of all Html elements and the methods to access all the html elements. It also defines the events for all the Html elements. With this object model javascript can change, modify, add and remove all the Html elements, attributes, CSS styling, on the webpage.

We can select Html elements using some DOM methods by giving class name or unique id to the html elements.
Some DOM selector method name are:
  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementsById()
  4. querySelector()
  5. querySelectorAll()

Event Bubble is a technique of DOM. Event Bubble propagating an event from the lowest to the upward on a DOM tree. It occurs when an Html element receive an event. The event bubble can read property of target html element of DOM tree.

Callback function is as type of function that call another function. That means a function that pass a function as argument to another function is known as callback function.
We Have to use function because:-
JavaScript executes code in a top-down, sequential basis. But still, there are some situations when code executes both sequentially and non-sequentially. This situation is known as asynchronous programming.
Callback function ensure that a function will not execute before the task is completed. Callback function is also use for doing same task multiple time.
